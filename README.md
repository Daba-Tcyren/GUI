Текствоый редактор для процессора

Название: Разработка текстового редактора с графическим интерфейсом
Цель работы: Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

Сведения об авторе: Тарбаев Даба-Цырен АП-327

Описание проекта: Разработан текстовый редаткор с базовым функионалом и графическим пользовательским интефейсом. Для удобства пользователя были реализованы дополнительные функции: работа с несколько файлами, подсветка синтаксиса, возможность изменения размеров текста выбор языка интерфейса приложения, нумерация строк, открытие файла при перетаскивании иконки в окно программы и т.п.

Используемые технологии: Язык пртограммирования: С#. Среда для графического интерфейса: Windows Forms. IDE: Visual Studio 2026.

Инструкция по сборке и запуску: 20 мб свободного места на твердотельном накопителе.
Точные шаги для установки зависимостей: Чтобы открыть редактор без IDE, нужно открыть GUI.exe.
Путь к готовому исполняемому файлу. Папка проекта->bin->Debug->GUI.exe


Описание интерфейса и функций (руководство пользователя): скриншоты работы программы и интерфейса с пояснением ключевых элементов с подробным указанием, как они вызываются (меню, кнопка, горячая клавиша) и что делают.

- Создать файл - Создается новый файл и открывается в приложение для работы с ним. (Ctrl+N)
 - Сохранить - Сохраняет текущий рабочий файл в приложении без указания пути до файла. (Ctrl+S)
 - Cохранить как - Открывается диалоговое окно для указания расположения, где сохраниться текущий рабочий файл. (Ctrl+Shift+S)
 - Выход - Совершается закрытие приложения. (Alt+F4)

 - Отмена - Отменяет совершенное действие. (Ctrl+Z)
 - Возврат - Возращает раннее отмененное действие. (Ctrl+Y)
 - Вырезать - Стирает выделенный текст и сохраняет в буфере обмене для вставки. (Ctrl+X)
 - Копировать - выделенный текст сохраняет в буфере обмене. (Ctrl+C)
 - Вставить - Текст из буфере обмене напишеться в окне редактирования. (Ctrl+V)
 - Удалить - Удаляет весь текст из окна редактирования. (Delete)
 - Выделить все - Выделяет весь текст из окна редактирования. (Ctrl+A)
